⚠️ REQUISITOS MÍNIMOS DO SERVIDOR:

• Ubuntu 22.04+ / Debian 11+ / CentOS 8+
• Node.js v18 ou superior instalado
• MySQL 8.0+ instalado e rodando

🔧 NOTA IMPORTANTE - UBUNTU 24.04:
O instalador detecta e corrige automaticamente o problema de autenticação
MySQL (auth_socket). Se você deixar a senha vazia quando solicitado,
uma senha será gerada automaticamente e salva no .env.
Nenhuma configuração manual é necessária!

🐛 TROUBLESHOOTING RÁPIDO:

Node.js não instalado:
  curl -fsSL https://deb.nodesource.com/setup_20.x | sudo -E bash -
  sudo apt-get install -y nodejs

MySQL não instalado:
  sudo apt-get update
  sudo apt-get install -y mysql-server
  sudo systemctl start mysql

Git não instalado:
  sudo apt-get install -y git

Porta 3001 ocupada:
  sudo lsof -ti:3001 | xargs kill -9
