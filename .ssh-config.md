## Deploy via SSH
```bash

scp -P 2224 -r ./dist/* flavio@31.97.64.43:/home/flavio/webapp/

# Ou usar rsync
rsync -avz -e "ssh -p 2224" ./dist/ flavio@31.97.64.43:/home/flavio/webapp/

# Reiniciar serviços
ssh -p 2224 flavio@31.97.64.43 "cd /home/flavio/webapp && pm2 restart ecosystem.config.cjs"
```
